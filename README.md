# Dashboard de Estadísticas - Frameworks de CSS 📊

Este proyecto es una implementación de un panel de control (Dashboard) profesional utilizando **Tailwind CSS** para la maquetación y **Chart.js** para la visualización dinámica de datos.

## 🚀 Características

*   **Diseño Moderno:** Interfaz limpia basada en componentes de tarjetas.
*   **Gráficas Dinámicas:** Visualización de crecimiento de usuarios e ingresos mensuales mediante Chart.js.
*   **Estadísticas en Tiempo Real:** Indicadores de rendimiento con variaciones porcentuales (positivas y negativas).
*   **Maquetación con Tailwind:** Uso extensivo de clases de utilidad para un diseño rápido y mantenible.

## 🛠️ Tecnologías Utilizadas

*   **HTML5** - Estructura semántica del documento.
*   **Tailwind CSS** - Framework de CSS orientado a utilidades para el diseño.
*   **Chart.js** - Librería de JavaScript para la creación de gráficas interactivas.

## 📂 Estructura del Proyecto

*   `index.html`: Archivo principal que contiene la estructura y la lógica de las gráficas.
*   `LICENSE`: Licencia MIT para el uso del código.
*   `README.md`: Documentación del proyecto.

## 💻 Cómo visualizar el proyecto

1.  Clona este repositorio o descarga los archivos.
2.  Abre el archivo `index.html` en cualquier navegador web moderno.
3.  ¡Listo! No requiere instalación de dependencias externas gracias al uso de CDNs.
